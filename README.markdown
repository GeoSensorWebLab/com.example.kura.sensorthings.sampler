# SensorThings Simulator App for Kura

This application simulates a sensor device and will submit data to a SensorThings instance.

For more information, please see the tutorial on [Github](https://github.com/GeoSensorWebLab/kura-sensorthings-tutorial) or [Bitbucket](https://bitbucket.org/geosensorweblab/kura-sensorthings-tutorial).
